# 💰 AI-Based Expense Manager

## 📌 Description

The AI-Based Expense Manager is a simple Python application designed to help users track daily expenses, manage budgets, and analyze spending patterns. It provides a user-friendly command-line interface to record expenses and view monthly summaries.

This project also incorporates basic AI concepts such as rule-based classification to automatically categorize expenses and generate smart spending insights.

---

## 🚀 Features

* Add daily expenses with amount, description, and date
* Automatic expense categorization (AI-based) 🤖
* View all recorded expenses
* Monthly expense summary
* Category-wise spending analysis
* Budget setting and tracking
* Smart alerts for overspending

---

## 🧠 AI Concept Used

This project uses basic Artificial Intelligence techniques such as:

* **Rule-Based Classification:** Automatically assigns categories (Food, Travel, Shopping, etc.) based on keywords in the description
* **Pattern Analysis:** Analyzes spending behavior and provides budget alerts

## ▶️ How to Run the Project

1. Clone the repository:

```

```

2. Navigate to the project folder:

```
cd expense-manager-ai
```

3. Run the program:

```
python main.py
```
